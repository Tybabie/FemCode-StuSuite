Student Suite

This project was built to help students and other prospective users achieve their aims by
tracking their expenses, having a to-do list and help them find books based on users' search

Expense Tracker

The Expense Tracker allows users to add their budget, expense name and expense amount And give a list of all the expenses entered

To-do List

The To-do List allows users to enter a list of things that they want to get done And users can remove an item from the list once it's done

Find a Book

Users enter a book name, or book info in the search box. Search result includes book title, book description and Authur(s) that matches the query
